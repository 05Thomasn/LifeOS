# LifeOS

LifeOS is a personal dashboard project I am building to learn front-end development, coding fundamentals, and eventually cloud-based application development.

The goal of LifeOS is to become an all-in-one personal command center for organizing tasks, goals, projects, calendar planning, study progress, and personal routines.

## Project Purpose

I started this as a summer side project to improve my coding skills while building something I would genuinely use. Instead of creating a random practice app, LifeOS is designed around my own workflow and interests.

The long-term idea is to build a clean, scalable productivity dashboard that can eventually include different life modules such as school, career planning, fitness, and personal goals.

## Current Progress

The project is currently in the early HTML/CSS stage.

So far, I have started building:

* A sidebar navigation layout
* A LifeOS logo area
* Navigation sections for Home, Dashboard, Tasks, Modules, Calendar, and Settings
* A basic color palette using CSS variables
* Initial styling for layout, icons, text, and sidebar structure

## Planned Features
